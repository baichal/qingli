# 文件清理工具

一个功能强大的文件清理工具，帮助您扫描、分类和清理个人文件。

## 功能特性

- 📁 **智能文件扫描** - 自动扫描文件系统，识别各类文件
- 🎯 **智能分类** - 内置多种文件分类规则
- ⚙️ **自定义规则** - 支持创建自定义分类规则
- 🧠 **AI学习** - 智能识别个人文件，支持学习功能
- 📋 **文件预览** - 支持文本、图片、视频预览
- 🗑️ **安全删除** - 支持删除文件到回收站或永久删除
- 📊 **可视化界面** - 美观的 Web 界面
- 🔄 **断点续传** - 支持扫描中断后继续扫描

## 系统要求

- Python 3.7+
- Windows 操作系统

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python app.py
```

程序会自动在浏览器中打开界面（http://127.0.0.1:8899

## 使用说明

### 扫描文件

1. 点击「开始扫描」按钮
2. 选择要扫描的驱动器
3. 等待扫描完成

### 管理文件

- 点击文件可预览内容
- 勾选文件后可删除
- 使用筛选功能快速找到目标文件

### 自定义规则

1. 点击「自定义分类规则」
2. 添加新规则，设置匹配条件
3. 测试规则是否生效

### 个人目录

1. 点击「个人目录」
2. 添加您认为是个人文件的目录
3. 这些目录下的文件会被特殊标记
